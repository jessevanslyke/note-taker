# Note Taker App

## Installation

```bash
npm i
```

## Run server

```bash
npm start
```

## Notes

This app will store and retrieve stored notes in a JSON file.

You can also delete any stored note in any order you wish, as the object with a key/value pair will delete the correct note.

The app will not refresh the page after a successful creation of a new note or deletion of a note at this time.

## User Story

AS A user, I want to be able to write and save notes

I WANT to be able to delete notes I've written before

SO THAT I can organize my thoughts and keep track of tasks I need to complete
